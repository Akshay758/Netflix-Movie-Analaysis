# Netflix Movies Analysis

## Project Overview
This project analyzes Netflix movie data to understand trends in movie releases, genres, ratings, and popularity. The goal of the project is to explore the dataset and extract insights that can help understand content distribution and viewer preferences on Netflix.

## Problem Statement
Streaming platforms like Netflix host thousands of movies and shows. Understanding the patterns in movie genres, release trends, and ratings can help identify what type of content performs better and what audiences prefer.

This project analyzes Netflix movie data to discover trends in movie releases, most common genres, and highly rated movies.

## Dataset
The dataset contains information about Netflix movies including:

- Movie Title
- Release Year
- Genre
- Duration
- Rating
- Country
- Director
- Cast

This dataset helps analyze how Netflix content is distributed across genres, years, and ratings.

## Tools and Technologies Used
- Python – Data analysis and data cleaning
- Pandas – Data manipulation and preprocessing
- Matplotlib / Seaborn – Data visualization
- Jupyter Notebook – Analysis environment

## Analysis Performed
The following analysis was performed:

- Number of movies released each year
- Most common movie genres on Netflix
- Distribution of movie ratings
- Top directors with the most movies
- Movie duration analysis
- Country-wise movie distribution

## Visualizations
Several visualizations were created to better understand the dataset:

- Movies released per year
- Genre distribution
- Rating distribution
- Top directors by movie count
- Movie duration distribution

## Key Insights
- Netflix has increased its movie releases significantly in recent years.
- Some genres such as Drama and Comedy appear more frequently than others.
- A small number of directors contribute multiple movies to the platform.
- Movie duration mostly falls within the typical feature-length range.

## Conclusion
Netflix movie analysis helps understand trends in streaming content. By analyzing movie data, we can identify popular genres, release patterns, and content distribution which can help in understanding audience preferences and content strategy.
